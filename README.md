# elisp
A small Lisp interpreter in JavaScript

![preview](/preview.png)

# Usage
```bash
yarn install && yarn start
```

# Test
```bash
yarn test
```

# Features
- read–eval–print loop (REPL)
- Mathematical Operators (+, -, *, /)
- Basic keywords: `do`, `if`, `def!`, `let*`, `fn*`
- Quote keywords: `quote`, `quasiquote`, `unquote`, `unquote-splicing`
